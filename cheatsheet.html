Aufbau der internen DSL

Eine neue Regel hinzufügen:

rule(
    Konditionen hier ...
    account = "Warenausgangsnummer"
)

Konditionen:

country = "Landname", // Land muss gleich dem Landnamen sein.
letter, // Sendung ist ein Brief.
parcel, // Sendung ist ein Paket.
bulky, // Sendung ist ein Sperrgut.
plz = "PLZ", // Postleitzahl fängt an mit PLZ.
dangerous [ = true | false ], // Sendung ist ein Gefahrgut. Der Wert true | false ist optional.
express [ = true | false ], // Wird per Expressversand versendet. Der Wert true | false ist optional.

Bitte beachte, dass bulky, dangerous und express explizit angegeben sein muss. Sonst wird automatisch verhindert, dass Sendunge mit diesen Charakteristiken in den Regeln erfasst werden.

Beispiel:

rule(
    country = "Deutschland", bulky, account = "47"
)

Aufbau der Rules-Klasse

/**
 * Erstellt eine neue Regel. Benötigt einen Warenausgang.
 *
 * @param {string} account Warenausgang
 * @returns {Rule} Die Regel
 */
function createNewRule(account);


class Rule {
    /**
     * Fügt eine Kondition hinzu, die nur von Briefen erfüllt wird.
     */
    function addLetterCondition();

    /**
     * Fügt eine Kondition hinzu, die nur von Paketen erfüllt wird.
     */
    function addParcelCondition();

    /**
     * Fügt eine Kondition für ein bestimmtes Land hinzu.
     *
     * @param {string} country Land
     */
    function addCountryCondition(string country);

    /**
     * Fügt eine Kondition hinzu, die immer erfüllt wird.
     */
    function addCatchAllCondition();

    /**
     * Testet, ob die Regel auf die gegebene Sendung / die Testeingabe zutrifft.
     *
     * @param {Delivery} delivery Sendung / Testeingabe
     * @return {bool} Trifft die Regel zu?
     */
    function test(Delivery delivery);
}

Aufbau der externen DSL

Unsere externe DSL versucht so stark wie möglich deutsche Sätz nachzustellen. Jede Regel fängt daher mit einem Sende an. Darauf folgen kann sowohl die Paketart einen Brief oder ein Paket oder schon eine Bedingung z. B. Gefahrengut. Zusätzlich dazu sind einige Artikel und Präpositionen erlaubt (z.B. einen oder von).

Bedingungen, die einen Wert benötigen, erwarten, dass der Wert direkt auf die Bedingung folgt: mit Land Deutschland oder mit PLZ 68251.

Verfügbare Bedingungen:

  1. Brief
  2. Expressversand
  3. Gefahrengut
  4. Land
  5. Maximalgewicht
  6. Paket
  7. PLZ
  8. Sperrgut
  9. Warenausgang

Verfügbare Artikel, Präpositionen und sonstige Worte:

  1. als
  2. ein
  3. eine
  4. einem
  5. einen
  6. im
  7. mit
  8. Sendung - Bedeutung: Deckt Paket, Brief und Sperrgut auf einmal ab
  9. und
  10. von
  11. zu
  12. zum

Mit den verfügbaren Wörtern könnte z.B. die folgenden Sätze gebildet werden.

Sende eine Sendung mit Gefahrengut und Land Österreich zum Warenausgang 45.
Sende ein Gefahrengut zum Warenausgang 39.
Sende einen Brief mit PLZ 12345 und Maximalgewicht von 15 kg zum Warenausgang 85.
